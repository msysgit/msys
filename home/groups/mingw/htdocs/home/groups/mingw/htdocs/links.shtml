MinGW

  • GNU Compiler Collection (GCC)
  • Laura Michaels' Mingw FAQ. Also see Laura's homepage for other useful information.
  • Colin Peters' site on MinGW -- this is where it all started.
  • J.J.vanderHeijden's GNU mingw32 tools. Contains some useful tools, and also old versions of GCC (2.8.x).
  • Jon Beniston <jon@sentinelware.freeserve.co.uk> is working on porting the Java runtime to Cygwin. If you are using the GNU Java front-end, you definitely want to visit http://www.cs.bris.ac.uk/~jb7216/libgcj/.
  • Robert Kern's instructions for Python Extensions with GCC/mingw32.
  • Mingw32 mailing list hosted by eGroups.COM.
  • Daniel Guerrero Miralles has some interesting items Mingw on his site. Mostly LCC-Win32 related, but a lot of that applies to Mingw as well.
  • PW32 is Paul Sokolovsky's effort to provide a POSIX-compliant C runtime library for Mingw.

Cygwin

  • Cygwin project. The mailing list archives are on-line, as is the Users Guide and so on. Please do look through these if you have a problem.

The MinGW FAQ contains various other useful links.
